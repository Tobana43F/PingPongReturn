### ゲーム概要

卓球のレシーブを題材にしたゲームです。  
サーブフォームから回転を見極め、コースを狙って返球し、ハイスコアを目指すことが目的のミニゲームです。

他の卓球ゲームであまり取りだたされていない「回転」に注目。  
「相手のサーブを見切って、コースを狙って返球する。」という卓球の気持ちよさを表現しました。
